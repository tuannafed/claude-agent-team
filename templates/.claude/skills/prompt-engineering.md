# Skill: Prompt Engineering

**Used by:** AI Engineer agent when designing LLM integration, system prompts, and user prompts.

---

## Scope

- System vs user prompt separation; few-shot vs zero-shot.
- Output format (JSON, markdown, structured) and validation.
- Token usage, temperature, and safety constraints.

## Conventions

- Document prompt templates in the track’s AI Engineer Output.
- Prefer deterministic formats (e.g. JSON schema) when the app consumes LLM output.
- Note cost and latency implications in the track.

See MANIFEST.md for when this skill is copied (AI/LLM team type or `--ai`).
