# agent-prompts/ — Reference Only

> These files are **plain markdown reference copies** of the agent prompts.
> They are NOT used directly by Claude Code.

## What to use instead

The active agent definitions live in:
```
templates/.claude/agents/*.md
```

Those files include YAML frontmatter (`name`, `model`, `tools`, `context: fork`) and are
automatically detected by Claude Code when copied to a project's `.claude/agents/` directory.

## Why this folder exists

Kept as human-readable reference for browsing agent content without opening YAML frontmatter files.
Content is identical to `.claude/agents/` counterparts.
